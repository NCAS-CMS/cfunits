cfunits.Units.units¶

Units.units¶

The units.

May be any string allowed by the units CF property.

See also

calendar

Examples:

>>> Units('kg').units
'kg'
>>> Units('seconds').units
'seconds'
>>> Units('days since 2000-1-1', calendar='366_day').units
'days since 2000-1-1'
