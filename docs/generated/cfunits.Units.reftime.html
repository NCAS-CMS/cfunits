cfunits.Units.reftime¶

Units.reftime¶

The reference date-time of reference time units.

See also

calendar, isreftime, units

Returns

cftime.datetime

Examples:

>>> u = Units('days since 2001-01-01', calendar='360_day')
>>> u.reftime
cftime.datetime(2001, 1, 1, 0, 0, 0, 0)
