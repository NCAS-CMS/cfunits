.. _Contributing:

**Contributing**
----------------

Contributions to cfunits are welcome and should be raised as issues in
the GitHub issue tracker: https://github.com/NCAS-CMS/cfunits/issues,
prior to submitting a pull request containing the new code made from a
fork of the cfunits GitHub code repository:
https://github.com/NCAS-CMS/cfunits.
